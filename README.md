# Financial Misinformation Detection System

## Problem
Investors are exposed to misleading financial information online.

## Solution
An AI-powered platform that analyzes financial content, verifies claims using RAG, and generates a credibility score.

## Features
- Financial sentiment analysis
- Fact verification
- Risk scoring
- Explainable AI outputs

## Tech Stack
- React
- FastAPI
- FinBERT
- LangChain
- Pinecone
- PostgreSQL
